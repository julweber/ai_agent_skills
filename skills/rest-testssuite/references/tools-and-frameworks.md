# Tools & Frameworks (Reference)

## Recommended Tools by Language

**JavaScript/TypeScript**
- Jest, Mocha, or Vitest
- Supertest or Axios for HTTP requests
- Nock for HTTP mocking
- Artillery or k6 for load testing

**Python**
- pytest or unittest
- requests or httpx
- responses or requests-mock for mocking
- locust for load testing

**Java**
- JUnit or TestNG
- RestAssured or MockMvc
- WireMock for mocking
- JMeter or Gatling for load testing

**Go**
- testing package
- httptest package
- testify for assertions
- vegeta for load testing

## Additional Tools
- Postman/Newman for API testing
- Swagger/OpenAPI validators
- OWASP ZAP for security testing
- Wiremock for service virtualization
- Docker for test environment isolation
