# REST API Testsuite Skill for Coding Agents

## Workflow
1. read the [Best Practices](references/best-practices.md) document
2. inspect the state of the current testsuite by running the full testsuite
3. identify and list the implemented REST endpoints of the application internally
4. inspect the directory structure of the test code
5. check the test coverage
6. Identify TODOs and list them using the according todo tool
7. Implement each of the points in the todo list by using an appropriate sub-agent
